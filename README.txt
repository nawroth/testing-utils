A package of  utilitiy classes useful for testing  projects built with
the Neo4j Graph Database.

This includes the  classes in the org.neo4j.test package  (and the sub
packages  of  this   package)  from  the  test  code   for  all  Neo4j
components. It  also contains all  classes named *TestAccess  from all
packages in the test code of all Neo4j components.
